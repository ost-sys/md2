Here are collected useful modules for Magisk! See if you can find something useful!

Magisk Manager

APK / ZIP application for installing and managing ROOT rights. Installed in a special modified version of Recovery (TWRP or Orange Fox Recovery Project)

Magisk Manager (MD3)

Magisk manager with new Android design (Material You)

Pixelify

Enabling all possible features from Google Pixel phones
Requirements:
1. Android Version: Android 7.0 to Android 13
2. Device Arm64
3. The presence of volume keys
4. Web for NGA Resources, Pixel Live Wallpapers, Device Personalization Services, and Pixel
5 Launcher. Zygisk or Riru (recommended but not required)

Universal Safety Fix

Magisk module to bypass Google SafetyNet attestation.

Ad Block Module

Magisk module for blocking ads in apps and on the web. Doesn't work in the YouTube app
